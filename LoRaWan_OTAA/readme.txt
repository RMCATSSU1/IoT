LoRaWan_OTAA.ino - The main code which we will use in Arduino.
decodeNodeUplink.js - The javascript code that we will use in the decoded payload in TTN.
viewTTNdata.py - The python script we will run to store TTN data on a txt file local to our PC.

The frequency band which we'll be using for OTAA is: United States 902-928 MHz, FSB 2 (used by TTN)

Please Note: This OTAA - over-the-air-activation code is for the RAK 3272S Breakout Board.
